House of Tiles Chatbot - Embed Example

This page demonstrates how to embed the House of Tiles AI chatbot into any website.

How to Embed the Chatbot

To embed the chatbot on your website, add the following code before the closing </body> tag:

<script>
  // Optional configuration
  window.HOUSEOFTILES_CHATBOT_CONFIG = {
    baseUrl: 'http://144.217.68.58:3023',       // URL where the chatbot widget is hosted
    apiUrl: 'http://144.217.68.58:8023',       // URL of the backend API
    primaryColor: '#bd1d73'                 // Primary color for the chatbot
  };
  
  // Load the widget script
  (function() {
    const script = document.createElement('script');
    script.src = window.HOUSEOFTILES_CHATBOT_CONFIG.baseUrl + '/widget.js';
    script.async = true;
    document.body.appendChild(script);
  })();
</script>
Note: When deploying to production, update the URLs to your actual hosted domain.
