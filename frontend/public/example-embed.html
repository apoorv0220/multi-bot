MRN Web Designs Chatbot - Embed Example

This page demonstrates how to embed the MRN Web Designs AI chatbot into any website.

How to Embed the Chatbot

To embed the chatbot on your website, add the following code before the closing </body> tag:

<script>
  // Optional configuration
  window.MRNWEBDESIGNS_CHATBOT_CONFIG = {
    baseUrl: 'https://your-frontend-domain',    // URL where the chatbot widget is hosted
    apiUrl: 'https://your-backend-domain',      // URL of the backend API
    tenantPublicKey: 'replace-with-widget-key'  // Per-tenant public embed key
  };
  
  // Load the widget script
  (function() {
    const script = document.createElement('script');
    script.src = window.MRNWEBDESIGNS_CHATBOT_CONFIG.baseUrl + '/widget.js';
    script.async = true;
    document.body.appendChild(script);
  })();
</script>
Note: When deploying to production, update the URLs to your actual hosted domain.
