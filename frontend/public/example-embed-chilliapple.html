Chatbot Embed Example - Chilliapple Tenant

This page demonstrates the public website embed for the Chilliapple tenant.

Update tenantPublicKey to the Chilliapple widget key that is mapped in WIDGET_EMBED_KEYS_JSON.

How to Embed

Add this snippet before your closing </body> tag:

<script>
window.CHATBOT_WIDGET_CONFIG = {
  baseUrl: "https://your-frontend-domain",
  apiUrl: "https://your-backend-domain",
  tenantPublicKey: "demo_pub_chilliapple_replace_me"
};
(function () {
  const script = document.createElement("script");
  script.src = window.CHATBOT_WIDGET_CONFIG.baseUrl + "/widget.js";
  script.async = true;
  document.body.appendChild(script);
})();
</script>
This is a mixed-source tenant (WordPress + static pages). The embed integration remains identical to other tenants; only the tenant public key differs.

Features

  • Tenant-scoped retrieval via widget key
  • Public visitor chat without auth flow
  • Mixed-source citations (website + blog) once indexed
