Chatbot Embed Example - Migraine Tenant

This page demonstrates the public website embed for the Migraine tenant.

How to Embed

Add this snippet before your closing </body> tag:

<script>
window.MRNWEBDESIGNS_CHATBOT_CONFIG = {
  baseUrl: "http://localhost:3043",
  apiUrl: "http://localhost:8043",
  tenantPublicKey: "demo_pub_migraine_4c97be2e",
  primaryColor: "#bf362e",
  position: "bottom-right"
};
(function () {
  const script = document.createElement("script");
  script.src = window.MRNWEBDESIGNS_CHATBOT_CONFIG.baseUrl + "/widget.js";
  script.async = true;
  document.body.appendChild(script);
})();
</script>
Public embeds should not require login and should not show admin shell/header. This page is intended to simulate that public usage pattern.

Features

  • Tenant-scoped retrieval via widget key
  • Public visitor chat without auth flow
  • Session persistence per widget key
